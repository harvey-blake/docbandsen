## 2. 预充手续费

### 点击充值

![充值](./img/13.jpg)

::: tip 仅支持 bsc 网络

- 转账到获取的钱包地址
- 可使用钱包转账
- 也可交易所提现至此地址
  :::

::: warning 请注意:

- 如手续费余额低于 10U 将无法添加策略
- 如手续费不够支付手续费,交易将暂停
  :::
