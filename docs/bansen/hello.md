# 申请币安 API 密钥

::: warning 请注意，创建 API 密钥前，您需要：

- 启用账户的双重身份验证（2FA）；
- 向现货账户充值任意金额即可激活账户；
- 完成身份认证。
  :::

1. **登录币安 App 并进入币安`专业版`，然后点击`【更多】`打开`【服务】`面板。**

![点击更多](./img/1.jpg)

2. **下拉至`【其他】`部分，然后点击`【API 管理】`图标。**

![api管理](./img/2.jpg)

3. **取消默认安全控制.**

![默认权限](./img/3.jpg)

4. **点击【创建 API】。**

![创建api](./img/5.jpg)

5. **选择所需的 API 关键类型。**<Badge text="选择系统生成" />

![选择API类型](./img/6.jpg)

6. **输入个人 API 的标签/名称。**

![设置标签](./img/7.jpg)

7. **编辑权限。**

![编辑权限](./img/8.jpg)

8. **勾选现货允许现货和杠杆交易并保存**

::: tip 保存二维码,添加 key 需要使用
:::

![编辑权限](./img/9.jpg)
